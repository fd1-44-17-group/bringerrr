Способы создать слайдер
  • 1.Вставить .jpg фаил в блок через бэкграунд и задать ему сайз 100% 100%
    • Плюсы: Занимает меньше времени на реализацию
    • Минусы: Нельзя использовать в качестве логотипа
  • 2.Нарисовать все исключительно с помощью градиентов
    • Плюсы: Повышение чувства собственой важности в случа успеха
    • Минусы: Занимает очень много времени
  • 3.Сконструировать с помощью 2ух png-шечек
    • Плюсы: Можно использовать в случаях, когда необходима картинка с резким переходом цвета
    • Минусы: ....
  • 4. Обойтись фигурами
    • Плюсы: можно сделать различные анимации(сборка из картикни из различных геометрических фигур)
    • Минусы: Займет много времени
  • 5.Нарисовать c помощью SVG-графики
    • Плюсы: Масштабирование производиться без потери качества
    • Минусы: Имеет определенные проблемы с воспроизведением в старых версия ИЭ
  • 6. Использовать нескольких jpg изображений
    манипулируя каскадностью форм в html, либо через z-index
    • Плюсы: ...
    • Минусы: должны сойтись определенные звезды на небе, чтобы получить файлы необходимых габаритов
  • 7.С помощью мнемоники: "Серп и молот" и "Звездочки"
    • Плюсы: Можно свободно манипулировать с помощью стилей
    • Минусы: ...
